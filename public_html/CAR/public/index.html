NOTE: You must use right-click. On an Apple tackpad, you can click with two fingers.
INSTRUCTIONS:

The display shows a map of the Netherlands with pickup points specified by x's and delivery points specified by o's. The depot is specified by a big star.

  1. Navigate to the CLUSTERING menu.
  2. Choose !SEEDS and then select the number of clusters to generate. The graph will now highlight multiple nodes in different colors. Clusters are then generated around these nodes.
  3. To create the clusters, select !CLUSTERS.
  4. Navigate to the ROUTING menu by selecting BACK then ROUTING.
  5. Select !TRIP and then a node of some color to create a tour of that cluster. Note that the full tour includes the depot. The circled node represents the first node visited in that cluster.
  6. To create a tour for all clusters, select !TRIP and then click the depot.
  7. Select CHANGE TRIP and then !ADD ADDR. to add an address to the current solution. Alternatively, select ADD ADDR. to choose both the address and the location in the tour where it should be added.
  8. Select !IMPROVE and then any cluster tour to try and improve the tour.
